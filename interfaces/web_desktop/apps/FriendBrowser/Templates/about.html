Welcome to the Friend Network!

The Friend Browser is your gateway to the Friend Network, an interconnected network of Friend nodes. This is not a web browser, in a normal sense, as the Friend Network is separate from the world wide web. Here, you will only be able to access content available in the Friend main net, or any subnets you or your administrator have preconfigured.

To read more about Friend Browser, please refer to the documentation.

Enjoy a new world of networked content using Friend Browser!
